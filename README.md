此项目是2016年做的demo，适用于移动端活动，跟后端交互的逻辑和JSSDK分享逻辑已经去掉。

## 游戏玩法说明

1. 填写手机号；
2. 在规定时间内，记忆牌面内容及位置；
3. 游戏中，点击任意两张卡牌，若相同则可以翻开，规定时间内全部配对成功即可过关；
4. 顺利过关可以获得一次抽奖机会，失败则可以关闭或者再玩一次；
